Method
------
For each of the 216 .dart files under lib/screens/, counted how many
OTHER .dart files under lib/ (excluding lib/archive/) import either
  - its relative path (e.g. "screens/coach/widgets/foo.dart"), or
  - its basename (e.g. "foo.dart").

A screen with zero hits is not imported by any other source file in
lib/ and is a strong orphan candidate. Note: this heuristic does NOT
count references from test/ or dynamic route tables that use strings;
a file with 0 hits may still be reachable via Navigator.pushNamed or
route generators, so treat the list as candidates, not proof.
